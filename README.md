# SI_AK

is a combination of village administration and financial applications to make it easier to access, this was created so that administration and financial activities can be carried out systematically and innovatively.

dalam bahasa indonesia "adalah penggabungan dari aplikasi administrasi dan keuangan desa agar lebih mudah dalam pengaksesannya, ini dibuat agar kegiatan administrasi dan keuangan dapat berjalankan secara sistemmatis dan inovatif."(thomas, 2025)
